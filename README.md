# Mini Chatbot Demo

Ein einfacher Chatbot, der mit dem vortrainierten Modell **DialoGPT-small** von Microsoft arbeitet.  
Das Modell generiert Antworten basierend auf dem bisherigen Gesprächsverlauf.

## Installation

1. Repository klonen
2. Virtuelle Umgebung erstellen: `python -m venv venv`
3. Abhängigkeiten installieren: `pip install -r requirements.txt`
4. Starten: `python chatbot.py` (Kommandozeile) oder `streamlit run streamlit_app.py` (Web)

## Verwendung

- Gib deine Nachricht ein und drücke Enter.
- Mit `exit` beendest du den Chat.
